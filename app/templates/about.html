关于本博客

这是一个基于 Flask 开发的个人技术博客，主要用于记录和分享我在网络安全和渗透测试领域的学习笔记、实践经验和技术见解。

博客特色

  • Typora 风格排版 - 清晰、优雅的阅读体验
  • Markdown 写作 - 专注于内容创作，支持代码高亮
  • 智能目录 - 自动生成文章目录，快速定位内容
  • 夜间模式 - 护眼的深色主题
  • 响应式设计 - 完美适配手机、平板和电脑
  • 代码复制 - 一键复制代码块
  • 文章导出 - 支持 Markdown 和 HTML 格式导出

技术栈

本博客采用现代化技术栈构建：

  • 后端：Flask 3.0 + SQLAlchemy + SQLite
  • 前端：Bootstrap 5 + 自定义 CSS
  • 内容：Python Markdown + Pygments
  • 特性：用户认证、分类标签、全文搜索、主题切换
